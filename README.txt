Ariel & Vai static wedding website

Open index.html in a browser to view the site.

Files:
- index.html: page markup
- styles.css: visual styling and responsive layout
- script.js: hash navigation, mobile menu, accordion behavior, and local RSVP storage
- assets/monet-garden-web.jpg: welcome page and banner artwork
- assets/story-photo.jpg: story page image

Notes:
- The site is completely static and has no backend.
- RSVP form submissions do not send anywhere. They are saved locally in the browser only.
- Replace placeholder hotel, registry, travel, and schedule text with final event details.
